# World Tour Music

Es una aplicación de escritorio que permita a los usuarios descubrir eventos
musicales, planificar rutas y compartir su experiencia de manera eficiente y
agradable.

## Pre-requisitos 📋
```
python 3.8.3

customtkinter 5.2.0

pillow 10.0.0

tkintermapview 1.29
```

## Instalación 🛠️
Clona el repositorio
